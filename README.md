EditNetWork
===========

技术支持：D3

功能实现：搭建自己的社会网络，在这里你可以上传自己的头像，编辑自己的圈子，类别、名称都是可以自定义的。操作历史的实时记录与
          网络信息的统计。

特色：即使你是一个新手，无需复杂的配置、分析方法，便可快速搭建属于你的社会网络。
